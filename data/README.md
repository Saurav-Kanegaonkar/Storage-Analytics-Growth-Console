# Data Sources

Synthetic ecommerce BI data for storage booking and growth experimentation.

- `markets.csv` - 20 market records
- `channel_daily.csv` - 12,000 market-channel-day performance rows
- `inventory_availability.csv` - 2,400 market-day capacity rows
- `competitor_prices.csv` - 1,600 competitor scrape rows
- `experiment_backlog.csv` - 18 growth experiment candidates
